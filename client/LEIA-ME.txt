===============================================
 ALBION MARKET INSIGHTS - CLIENTE v2.0
 Contribuicao com dados de mercado
===============================================


O QUE E?
--------
  Este programa captura os precos que voce ve
  no mercado do Albion Online e envia pro
  servidor da guild, ajudando todo mundo a
  ter dados de mercado atualizados.


COMO INSTALAR (passo a passo):
------------------------------

  PASSO 1 - DESCOMPACTAR
  =======================
  Descompacte este arquivo numa pasta.
  Pode ser qualquer pasta, ex:
    C:\Users\SeuNome\Documents\albion-client\


  PASSO 2 - INSTALAR NPCAP
  =========================
  O Npcap e necessario para capturar os dados
  de rede do jogo.

  a) Execute o arquivo npcap-1.88.exe que vem
     nesta pasta

  b) Na tela de instalacao, MARQUE a opcao:
     [x] WinPcap API-compatible Mode
     (isso fica na ultima tela do instalador)

  c) Clique em Install e aguarde

  d) Se pedir pra reiniciar, reinicie o PC


  PASSO 3 - CONFIGURAR
  =====================
  a) Execute o arquivo CONFIGURAR.bat
     (clique com botao direito > Executar como
     administrador)

  b) Digite seu nome/gamertag

  c) Digite sua API Key
     (pegue no painel do site: Contributors)

  d) Pronto! O config.json sera criado


  PASSO 4 - INICIAR
  ==================
  a) Abra o Albion Online e entre no jogo

  b) MINIMIZE o jogo (nao feche)

  c) Execute INICIAR.bat
     IMPORTANTE: clique com botao DIREITO >
     "Executar como administrador"

  d) Volte pro jogo e va ate o mercado

  e) Clique nos itens pra ver os precos
     (abra pelo menos 1 pagina de precos)

  f) Os dados sao enviados automaticamente!


COMO FUNCIONA POR DENTRO:
-------------------------
  1. O client ficar escutando os pacotes de rede
     que o jogo envia/recebe (porta UDP 5056)

  2. Quando voce abre o mercado no jogo, o client
     captura os precos que o servidor manda

  3. A cada 5 segundos, os dados capturados sao
     enviados pro servidor em lotes

  4. No servidor, os dados ficam disponiveis pra
     todo mundo no site


O QUE E ENVIADO?
----------------
  Apenas dados de mercado publicos:
  - Item ID (ex: T4_BAG)
  - Cidade (ex: Caerleon)
  - Precos (sell/buy min/max)
  - Qualidade do item
  - Timestamp

  NAO e enviado:
  - Sua senha
  - Seus itens
  - Sua localizacao
  - Qualquer dado pessoal


REQUISITOS:
------------
  - Windows 10 ou 11 (64-bit)
  - Npcap instalado (WinPcap compat mode)
  - Internet ativa
  - Permissoes de administrador
  - Albion Online rodando


PROBLEMAS COMUNS:
-----------------

  "Nao consigo executar como administrador"
    -> Clique com botao DIREITO no .bat
    -> Selecione "Executar como administrador"
    -> Nao e so dar duplo clique!

  "O client abre e fecha rapido"
    -> Provavelmente nao esta rodando como admin
    -> Execute novamente como administrador

  "ERRO: Npcap nao encontrado"
    -> Instale o npcap-1.88.exe que esta na pasta
    -> Marque "WinPcap API-compatible Mode"
    -> Reinicie o PC depois de instalar

  "ERRO: API key invalida"
    -> Execute CONFIGURAR.bat
    -> Cole sua API key corretamente
    -> A key fica no painel do site

  "Nao aparece nada no client"
    -> Verifique se o Albion Online esta aberto
    -> Va ate o mercado no jogo
    -> Clique em alguns itens pra carregar precos

  "O jogo trava com o client aberto"
    -> O client nao interfere no jogo
    -> Ele so le os pacotes de rede
    -> Se travar, feche e reabra o client
