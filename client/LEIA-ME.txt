ALBION MARKET INSIGHTS - CLIENTE v2.0
======================================

O que e?
  Cliente que captura dados de mercado do Albion Online
  e envia pro servidor Albion Market Insights.


INSTALACAO (3 passos):
-----------------------

1. INSTALE O NPCAP
   Baixe: https://npcap.com/#download
   Marque: "WinPcap API-compatible Mode"
   Reinicie o computador apos instalar

2. CONFIGURE
   Execute CONFIGURAR.bat
   Digite seu nome e sua API Key
   (pegue a API Key no painel do site)

3. INICIE
   Execute INICIAR.bat
   IMPORTANTE: clique com botao direito →
   "Executar como administrador"


COMO FUNCIONA:
--------------
  1. Voce abre o Albion Online
  2. Vai ate o mercado de qualquer cidade
  3. Clica nos itens pra ver os precos
  4. O cliente captura esses dados automaticamente
  5. Envia pro servidor em lotes a cada 5 segundos


REQUISITOS:
------------
  - Windows 10/11 (64-bit)
  - Npcap instalado
  - Permissoes de administrador
  - Internet ativa


TROUBLESHOOTING:
----------------
  "Npcap nao encontrado"
    → Instale o Npcap de https://npcap.com
    → Marque a opcao WinPcap compatible

  "Sem permissoes"
    → Clique com botao direito no INICIAR.bat
    → Executar como administrador

  "API key invalida"
    → Execute CONFIGURAR.bat e coloque sua key
    → A key fica no painel do site (Contributors)
